Theory Reference

Derjaguin Approximation (Proximity Force Approximation)

For two curved surfaces at separation D ≪ R, the interaction force relates to the flat–flat free energy per unit area G(D) via:

F(D) = 2π Reff · G(D)

For non-retarded vdW, G(D) = −A/(12πD²), giving:

GeometryReffEDAValid when
Sphere–Sphere (equal R)R/2−AR / (12D)D ≪ R
Sphere–FlatR−AR / (6D)D ≪ R

Hamaker Continuum (reference)

Treating matter as continuous with uniform number density ρ and integrating −C₆/r⁶ analytically gives closed-form results (the Hamaker 1937 approach, A = π²ρ²C₆). For two equal spheres (s = 2R + D):

E = −(A/6) [ 2R²/(D(D+4R)) + 2R²/s² + ln( D(D+4R)/s² ) ]

For sphere–flat:

E = −(A/6) [ R/D + R/(D+2R) + ln( D/(D+2R) ) ]

This is the reference line (dotted). It breaks down at D ≲ Δ (atom spacing) where the discrete surface structure matters — a regime the DA is already useless in.

Bead Model (pairwise atomic sum)

Each sphere is packed with point atoms on a 3D Cartesian grid (spacing Δ ≈ atom diameter). The total vdW attraction is the explicit pairwise double sum over all cross-object pairs:

Enum = Σi∈obj1 Σj∈obj2 −C₆/rij⁶

C₆ is calibrated against the Hamaker continuum at a reference separation Dref = R, so the bead sum converges toward the continuum result as N → ∞ (Δ → 0). The key distinction is geometric: the bead sum directly computes the true pairwise interaction for the discrete geometry, while the DA applies only a proximity-force geometric shortcut. As D/R grows the DA overestimates attraction; the bead sum doesn't.

Bead brightness encodes proximity to the opposite object — atoms nearest the gap glow hottest, showing where the interaction is concentrated.
